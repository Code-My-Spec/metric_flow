## Issues

### No feature gate or require_subscription plug implemented

#### Severity

HIGH

#### Scope

APP

#### Description

Story 543 requires a `require_subscription` plug or LiveView on_mount hook that checks whether the current account has an active subscription. This hook needs to be applied to the correlations, insights, and visualizations routes in the router. Free users should see a paywall overlay or be redirected to `/subscriptions/checkout`. The `Billing.Subscription` schema exists but no query function checks subscription status for the current scope, and no router-level gate enforces it.
