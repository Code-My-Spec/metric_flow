## Issues

### Reports page crashes with KeyError on chart_type

#### Severity
HIGH

#### Description
`/reports` crashed with `KeyError: key :chart_type not found` at `report_live/index.ex:164`. The template accessed `report.chart_type` but the `Visualization` schema has no `chart_type` field — it's stored inside the `vega_spec` map as `vega_spec["chart_type"]`. Fixed by changing to `get_in(report.vega_spec, ["chart_type"])`.

### Rolling review metric calculation not implemented

#### Severity
HIGH

#### Description
The story requires three computed metrics: `BUSINESS_REVIEW_DAILY_COUNT`, `BUSINESS_REVIEW_TOTAL_COUNT`, and `BUSINESS_REVIEW_AVERAGE_RATING`. None of these exist in the codebase. No aggregation/rolling calculation module exists. The raw `review_count` and `review_rating` metrics are synced from GBP but never transformed into the rolling metrics the story describes. This is the core feature of story 515 and it is missing entirely.
