Metric Flow Design System

Custom dark theme with aurora background, floating cards, and subtle luminous borders

Theme

Custom DaisyUI theme: metricflow

Characteristics

  • Base: Dark mode, deep neutral background
  • Surface: Semi-transparent panels with backdrop blur
  • Borders: Subtle luminous edges, glow on hover
  • Background: Animated aurora gradient (green, indigo, cyan)
  • Corners: 8px (medium rounded)
  • Density: Balanced spacing

Usage in Phoenix

<!-- root.html.heex -->
<html data-theme="metricflow">

<!-- tailwind.config.js -->
daisyui: {
  themes: ["metricflow"]
}

Tone

Professional with a premium, futuristic edge. Data-confident, not flashy.

Visual Personality

  • DensityBalanced
  • Corners8px (medium)
  • GlowSubtle — visible on hover
  • FeelPremium analytics
  • InspirationNorthern lights + Cyberpunk (dialed back)

Copy Guidelines

  • Use direct, confident language — "Connected" not "Successfully connected!"
  • Metrics speak for themselves — minimize adjectives around numbers
  • Error states are helpful, not alarming — guide toward resolution
  • Button labels are action verbs: "Connect", "Sync", "Export", "Remove"
